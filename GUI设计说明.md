# YouTube 批量下载器 - GUI 设计说明

## 🎨 界面优化内容

### 1. 整体布局
- **窗口尺寸**: 1000x750 (可调整，最小 800x600)
- **圆角设计**: 所有面板使用 15px 圆角，更现代化
- **配色方案**:
  - 深色模式：深蓝色主题
  - 浅色模式：清新蓝白配色
- **间距优化**: 统一的内外边距，视觉更舒适

### 2. 顶部标题区
```
🎬 YouTube 批量下载器
支持单个视频、批量下载和播放列表 | 多清晰度选择 | 实时进度显示
```
- 大标题 (28px 粗体)
- 副标题说明功能特性
- 渐变色文字效果

### 3. URL 输入区
```
📝 输入 YouTube URL  (每行一个链接，支持视频和播放列表)
┌─────────────────────────────────────────┐
│                                         │
│  [输入框 - 120px 高度]                  │
│                                         │
└─────────────────────────────────────────┘
```
- 图标 + 标题
- 灰色提示文字
- 圆角边框 (10px)
- 2px 边框高亮

### 4. 控制面板
```
左侧：
🎥 清晰度: [1080p ▼]    📁 保存到: ~/Downloads/YouTube [选择目录]

右侧：
[▶ 开始下载]  [⏹ 停止]
```
- 分左右两区
- 图标 + 文字标签
- 按钮使用语义化颜色：
  - 开始：绿色 (#2fa572)
  - 停止：红色 (#d32f2f)
  - 选择目录：蓝色 (#3b8ed0)
- 按钮高度 35-40px，圆角 8-10px

### 5. 统计面板
```
📊 准备就绪 - 请输入 YouTube URL 开始下载
```
下载时显示：
```
⬇️ 总计: 3 | ⏳ 等待: 1 | ⬇️ 下载中: 2 | ✅ 完成: 0 | ❌ 失败: 0
```
- 浅蓝色背景
- 动态 emoji 指示器
- 实时更新统计

### 6. 下载列表
```
📥 下载列表

┌─────────────────────────────────────────┐
│ 🎬 Me at the zoo                        │
│ ████████████░░░░░░░░░░░░░░░░░░░░ 40%   │
│ ⬇️ 下载中... 40% | 速度: 500KB/s | 剩余: 00:05 │

┌─────────────────────────────────────────┐
│ 🎬 Another Video                        │
│ ░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░ 0%    │
│ ⏳ 等待中...                            │
└─────────────────────────────────────────┘
```

每个下载项：
- 白色卡片 (深色模式为深灰)
- 12px 圆角
- 2px 边框
- 包含：
  - 🎬 视频标题 (13px 粗体)
  - 进度条 (20px 高，10px 圆角)
  - 状态信息 (11px，带颜色)

### 7. 状态颜色系统

| 状态 | Emoji | 颜色 | 进度条颜色 |
|------|-------|------|-----------|
| 等待中 | ⏳ | 灰色 | 蓝色 |
| 下载中 | ⬇️ | 蓝色 | 绿色 |
| 完成 | ✅ | 绿色 | 绿色 |
| 失败 | ❌ | 红色 | 红色 |
| 已取消 | ⏹ | 橙色 | 橙色 |

### 8. 空状态
当没有下载任务时显示：
```
暂无下载任务

请在上方输入 YouTube URL 并点击"开始下载"
```
- 居中显示
- 灰色文字
- 50px 上下边距

### 9. 交互反馈
- **按钮悬停**: 颜色加深效果
- **禁用状态**: 半透明显示
- **进度条**: 平滑动画过渡
- **状态更新**: 实时刷新，无闪烁

### 10. 响应式设计
- 窗口可调整大小
- 最小尺寸限制 (800x600)
- 下载列表自动滚动
- 长文本自动截断

## 🎯 设计原则

1. **清晰直观**: 使用 emoji 和图标增强识别度
2. **视觉层次**: 通过大小、颜色、间距区分重要性
3. **状态反馈**: 每个操作都有明确的视觉反馈
4. **现代美观**: 圆角、阴影、渐变等现代设计元素
5. **易用性**: 大按钮、清晰标签、合理布局

## 🚀 使用体验

1. **启动**: 双击启动脚本，窗口快速打开
2. **输入**: 粘贴 URL，支持多行
3. **配置**: 选择清晰度和保存位置
4. **下载**: 点击开始，实时查看进度
5. **完成**: 绿色勾号标记，统计面板显示结果

启动应用后，你会看到：
- 顶部：标题和说明
- 中上：URL 输入框
- 中间：控制按钮和设置
- 中下：统计信息
- 底部：下载列表（可滚动）

所有元素都经过精心设计，提供最佳的用户体验！
