# YouTube 批量下载器 - 使用说明

## 快速开始

### macOS 用户

1. **双击运行** `启动应用.command` 文件
2. 如果提示权限问题：
   - 右键点击文件
   - 选择"打开"
   - 在弹出对话框中再次点击"打开"确认

### Windows 用户

1. **双击运行** `启动应用.bat` 文件

---

## 首次运行

首次运行时，程序会自动：
1. 创建虚拟环境
2. 安装所需依赖（yt-dlp、customtkinter 等）
3. 启动 GUI 应用

**注意**：首次运行可能需要 1-2 分钟安装依赖，请耐心等待。

---

## 使用方法

### 1. 输入视频 URL

在顶部文本框中输入 YouTube 视频链接，支持：
- **单个视频**：直接粘贴视频链接
- **批量下载**：每行一个链接
- **播放列表**：粘贴播放列表链接，自动下载所有视频

**示例**：
```
https://www.youtube.com/watch?v=jNQXAC9IVRw
https://www.youtube.com/watch?v=dQw4w9WgXcQ
https://www.youtube.com/playlist?list=PLxxxxxx
```

### 2. 选择清晰度

从下拉菜单中选择视频清晰度：
- **最佳质量**：自动选择最高质量
- **1080p**：全高清（推荐）
- **720p**：高清
- **480p**：标清（文件较小）
- **仅音频**：只下载音频（MP3/M4A）

### 3. 选择保存位置

- 点击"选择目录"按钮
- 选择视频保存位置
- 默认保存到：`~/Downloads/YouTube`

### 4. 开始下载

- 点击绿色的"开始下载"按钮
- 下载列表会显示每个视频的：
  - 视频标题
  - 下载进度条
  - 下载速度
  - 剩余时间
  - 状态（等待中/下载中/完成/失败）

### 5. 停止下载

- 点击红色的"停止"按钮可以取消所有下载

## 常见问题

### Q1: 双击启动脚本没反应？

**macOS 解决方法**：
1. 右键点击 `启动应用.command`
2. 选择"打开"
3. 在弹出对话框中点击"打开"确认

或者在终端中运行：
```bash
cd youtube_downloader
bash 启动应用.command
```

### Q2: 提示 "Tkinter 未安装"？

**macOS 解决方法**：
```bash
brew install python-tk@3.13
```

然后删除 `venv` 文件夹，重新运行启动脚本。

### Q3: 下载速度很慢？

- 检查网络连接
- 尝试选择较低清晰度（如 480p）
- 某些视频可能受地区限制

### Q4: 下载失败？

可能原因：
- 视频已被删除或设为私密
- 视频有版权保护
- 网络连接问题
- YouTube API 变化（需要更新 yt-dlp）

**解决方法**：
```bash
source venv/bin/activate
pip install --upgrade yt-dlp
```

### Q5: 如何更新程序？

更新 yt-dlp 到最新版本：
```bash
cd youtube_downloader
source venv/bin/activate
pip install --upgrade yt-dlp
```

## 技术信息

### 系统要求
- **操作系统**：macOS / Windows / Linux
- **Python**：3.8 或更高版本
- **网络**：稳定的互联网连接

### 核心依赖
- **yt-dlp**：YouTube 下载引擎
- **customtkinter**：现代化 GUI 框架
- **Pillow**：图像处理库

### 项目结构
```
youtube_downloader/
├── 启动应用.command    # macOS 启动脚本
├── 启动应用.bat        # Windows 启动脚本
├── main.py            # 应用入口
├── requirements.txt   # 依赖列表
├── gui/               # GUI 界面代码
├── core/              # 核心下载逻辑
└── venv/              # 虚拟环境（自动创建）
```
