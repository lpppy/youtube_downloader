## 📖 使用方法

### 快速开始

1. **启动应用**
   ```bash
   # macOS
   双击 "启动应用.command"

   # Windows
   双击 "启动应用.bat"

   # Linux
   bash start.sh
   ```

2. **输入 URL**
   - 粘贴 YouTube 视频链接
   - 每行一个
   - 支持播放列表

3. **选择设置**
   - 清晰度：1080p（推荐）
   - 保存位置：自定义

4. **开始下载**
   - 点击"▶ 开始下载"
   - 查看实时进度
   - 等待完成

## 🔧 故障排除

### 问题 1：双击启动无反应
**解决方法**：
- macOS：右键 → 打开 → 确认
- 或在终端运行：`bash 启动应用.command`

### 问题 2：提示 Tkinter 未安装
**解决方法**：
```bash
brew install python-tk@3.13
rm -rf venv
bash 启动应用.command
```

### 问题 3：下载失败
**可能原因**：
- 网络问题
- 视频被删除
- 版权限制

**解决方法**：
- 检查网络连接
- 更新 yt-dlp：`pip install --upgrade yt-dlp`
- 查看错误日志
